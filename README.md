# try-arithmetic-operations-csharp
[C#] 四則演算 (数値, +, -, *, / のみの演算)

### 環境
- vscode 1.47
- dotnetcore 3.1.3
- NUnit 3.12.0

### Run
```bash
$ cd TryArithmeticOperations
$ dotnet run
```

### Test
```bash
$ cd CalculatorService.Tests
$ dotnet test
```
